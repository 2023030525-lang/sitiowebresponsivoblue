Introducción a la carrera

La Ingeniería en Tecnologías de la Información forma profesionales capaces de desarrollar soluciones tecnológicas innovadoras, integrando software, redes y servicios digitales para optimizar los procesos de una organización. La asignatura de Programación Web permite extender la disponibilidad y alcance de los servicios mediante aplicaciones en línea.

Programa Educativo

Este curso pertenece al programa educativo de Ingeniería en Tecnologías de la Información / Ingeniería en Sistemas Computacionales.

Cuatrimestre: Séptimo

Propósito de Aprendizaje

El alumno implementará sistemas empleando tecnologías web para extender la disponibilidad y alcances de los servicios de la organización.

Unidades de Aprendizaje

  • I. Introducción a los sistemas web
  • II. Programación del lado del cliente
  • III. Programación del lado del servidor
  • IV. Autenticación de usuarios

Temas por Unidad

Unidad I: Introducción a los sistemas web

Modelo cliente-servidor, HTTP/HTTPS, TCP/IP, arquitectura web, instalación y configuración de servidores.

Unidad II: Programación del lado del cliente

HTML, CSS, JavaScript, documentos dinámicos y diseño responsivo.

Unidad III: Programación del lado del servidor

Lenguajes como PHP, Python, manejo de sesiones, APIs, servicios web y monitoreo del sistema.

Unidad IV: Autenticación de usuarios

Seguridad en servidores web, manejo de excepciones y mecanismos de autenticación.

Nota: Se recomienda utilizar W3Schools para reforzar conocimientos de desarrollo web.
